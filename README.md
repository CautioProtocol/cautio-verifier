# Cautio Verifier – Offline Proof Validation

**Cryptographic integrity check. Runs 100% offline in your browser.**

### Quick Start
1. Download or view: https://brettscode-711.github.io/cautio-verifier/
2. Open verifier.html in any browser (Chrome, Safari, Firefox)
3. Drag & drop a .proof.json file
4. See instant VALID / INVALID result + Merkle path
5. Export PDF report for records

### Key Features
- Fully client-side: No uploads, no tracking, no server calls
- Verifies Merkle inclusion proofs generated by Cautio Engine
- Supports SHA-256 and Keccak-256
- Dark-mode cyber aesthetic
- MIT licensed – free for any use

### Privacy & Security
Runs entirely in your browser. No data is transmitted anywhere.
